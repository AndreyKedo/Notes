## Условные операторы


## Методы массивов
* `length` возвращает длину массива с названием `array` (также подходит для строк)
```hbs
{{ array|length }}
```
* `length_is` возвращает логическое значение `true/false`, если длина массива `array` совпадает с указанной (например, `3`)
```hbs
{{ array|length_is:"3" }}
```
